# BookStore Application

## Overview
This project implements a simple graphical BookStore application using Java Swing. The system allows an owner to manage books and customers, while registered customers can log in, purchase books, redeem points, and view their membership status.

The application uses a single-window GUI design, file storage for book and customer data, UML design diagrams, and the State Design Pattern to manage customer membership levels. The system was designed, implemented, and demonstrated as a group project for COE528.

## Features
- Owner login with admin credentials
- Owner can add and delete books
- Owner can add and delete customers
- Customer login system
- Customers can buy selected books
- Customers can redeem points toward purchases
- Silver and Gold customer status system
- State Design Pattern used for customer membership status
- Data saved and loaded using books.txt and customers.txt
- Single-window Java Swing GUI
